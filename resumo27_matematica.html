Dominando Funções Trigonométricas

Um guia visual para decifrar os parâmetros, transformações e aplicações de senoides e cossenoides nos vestibulares.

Decifrando a Função Completa

A maioria dos problemas de vestibular usa uma forma geral da função trigonométrica. Entender o que cada parâmetro faz é a chave para resolver as questões.

f(x) = A + B ⋅ sen(C ⋅ x + D)

A: Deslocamento Vertical

Move o gráfico para cima (A > 0) ou para baixo (A < 0). É a "nova linha média" da onda.

B: Amplitude

Determina a "altura" da onda a partir da linha média. A amplitude é sempre |B|.

C: Período

Altera a frequência. O período (P) é calculado por P = 2π / |C|.

D: Deslocamento Horizontal

Move o gráfico para a esquerda ou direita (defasagem).

Visualizando as Transformações Gráficas

Amplitude: f(x) = 2sen(x)

O parâmetro 'B' estica o gráfico verticalmente.

Período: f(x) = sen(2x)

O parâmetro 'C' comprime o gráfico horizontalmente.

Desl. Vertical: f(x) = sen(x) + 1

O parâmetro 'A' move o gráfico para cima ou para baixo.

Desl. Horizontal: f(x) = sen(x - π/2)

O parâmetro 'D' move o gráfico para os lados.

Estratégias de Resolução

Encontrando Máximos e Mínimos

Para achar o valor máximo, use sen/cos = 1. Para o mínimo, use sen/cos = -1.

Para saber **quando** ocorre, iguale o argumento da função a π/2 (máx. do seno) ou π (mín. do cosseno).

Modelando um Problema

"Traduza" o enunciado: A altura média de algo é o **deslocamento vertical (A)**. A variação a partir da média é a **amplitude (B)**. O tempo para um ciclo completo define o **período (P)**, que permite achar **C**.

Cancelamento de Ruído (Defasagem)

Para cancelar uma onda y = B⋅sen(Cx), é preciso gerar uma onda oposta, -y. Lembre-se da identidade: -sen(θ) = sen(θ + π). Basta somar π ao argumento.
